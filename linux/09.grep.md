# grep (Global Regular Expression Print)

- 匹配 `file` 文件中含有 `text` 的行并打印

```shell
grep "text" file
```

- 显示匹配的行号

```shell
grep -n "text" file
```

- 反向匹配（不包含 `text` 的行）

```shell
grep -v "text" file
```

- 显示匹配成功的次数

```shell
grep -c "text" file
```

- 仅对参数列表里的文件名进行匹配

```shell
grep -l "text" file*
```

- 忽略大小写

```shell
grep -i "TeXt" file
```

- 严格匹配

```shell
grep -x "line" file
```

匹配内容只有 `line` 的行

- 从文件里读取要匹配的文本

```shell
grep -f text_file file
```
